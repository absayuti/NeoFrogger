# NeoFrogger

A "Frogger" game clone written in *NeoBASIC* for *Neo 6502*.

The files are:

1. **Neofrogger.neo** -- source code of the program, written as structured BASIC in plain text (using NotePad++).
2. **NeoFrogger.bas** -- tokenized version of the program. Was converted from *.neo* to *.bas* using the following command: *python3 makebasic.zip NeoFrogger.neo -oNeoFrogger.bas*
3. **neofrogger.gfx** -- graphics file containing sprite and tile images
4. **sprite_16.png** -- 16x16 sprite images
5. **tile_16.png** -- 16x16 tile images

To run the program:

1. Put the files **NeoFrogger.bas** and **neofrogger.gfx** in the **storage** folder of either your **Neo6502 SD card**; or your *working directory* if your using the *Neo emulator*.
2. Load and run **NeoFrogger.bas**
